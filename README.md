# 💚 GoHealthy

A modern, responsive, and user-friendly web platform designed to manage and respond to health-related requests such as blood donations, emergency help, and medical support.

---

## 🚀 Project Overview

GoHealthy is a lightweight full-stack web application built without any database, focusing on simplicity, speed, and accessibility.

It allows users to:

* View health-related requests
* Submit new requests
* Help others by marking requests as completed
* Manage requests with edit and delete functionality

---

## 🎯 Purpose

The main goal of GoHealthy is to:

* Provide a quick platform for urgent medical help
* Connect people who need help with those who can provide it
* Keep the system simple and fast without complex setups

---

## 🛠️ Tech Stack

**Frontend:**

* HTML

**Backend:**

* Node.js

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/gohealthy.git
cd gohealthy
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the server

```bash
node server.js
```

### 4. Open in browser

```
http://localhost:3000/dashboard.html
```

---

## 🧪 How to Use

1. Go to **Add page** and create a request
2. View all requests on the **Requests page**
3. Mark requests as completed
4. Edit or delete any request
5. Check stats on the **Dashboard**
